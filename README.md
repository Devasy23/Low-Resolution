# Dependencies

## Frontend (React, NodeJS)

### Install dependencies using
```
npm install

# OR, if you are in the root directory
cd frontend && npm install
```

### Deploy using
```
npm start

# OR, if you are in the root directory
cd frontend && npm start
```

### Configuration
See [./frontend/.env](./frontend/.env) for configuration options
